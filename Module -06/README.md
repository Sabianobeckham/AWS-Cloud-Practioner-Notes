# Module 6 – AWS Cloud Storage

## Date Completed
24 June 2026

## Topics Covered

- Amazon Elastic Block Store (EBS)
- Amazon S3
- Amazon EFS
- Amazon FSx
- S3 Storage Classes
- AWS Storage Gateway
- AWS Elastic Disaster Recovery

## Key Takeaways

- Learned the differences between block, object, and file storage.
- Understood how Amazon S3 stores data using buckets and objects.
- Learned the different S3 storage classes and when to use each.
- Explored Amazon EFS and Amazon FSx for shared file storage.
- Learned how AWS Storage Gateway connects on-premises infrastructure with AWS.
- Understood how AWS Elastic Disaster Recovery improves business continuity.

## Reflection

This module helped me understand that choosing the right storage service depends on the workload, cost, performance, and availability requirements instead of using one storage service for everything.
